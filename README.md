# Product Recommender in Flutter

This is a Flutter-based frontend application for a smart product recommendation system.

## Features
- Firebase Authentication
- User login & signup
- Product listing
- Product recommendation UI
- CRUD operations

## Tech Stack
- Flutter
- Dart
- Firebase
- REST APIs

## Project Scope
This project aims to build a smart product recommender frontend integrated with Firebase and backend APIs as part of a Final Year Project (FYP).
